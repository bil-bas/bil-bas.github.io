Releasy

Updated Wednesday, 1st February 2012

v0.2.2 (Beta)

Releasy automates the release of Ruby applications (from: “release easy”). It generates a number of Rake tasks for use when there is a need to release a new version of the application.

Installation

> gem install releasy
